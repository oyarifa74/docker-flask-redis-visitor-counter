# Docker Visitor Counter App

## Overview
A beginner-friendly Docker project that containerizes a Flask web app and Redis using Docker Compose.

## Tech Stack
- Docker
- Docker Compose
- Python Flask
- Redis
- Git

## Features
- Multi-container setup
- Dockerfile-based app image
- Compose orchestration
- Redis volume persistence
- Health checks
- Environment variables

## Run the project
```bash
docker compose up --build
